控制平面

设备接入、状态可见、版本发布统一归口

面向小而稳的基础设施控制平面，聚焦 Agent 安装、设备状态巡检与安装包发布，保持流程清晰、界面克制、操作可追踪。

安装流程

01

生成一次性注册 token，按平台输出推荐安装命令，适合首批接入和标准化部署。

设备视图

02

按状态、系统、架构和版本筛选 Agent，快速定位异常节点和待维护对象。

发布管理

03

安装包上传后自动校验，支持多 channel 发布与退役，维持发布来源和版本边界。

运维模型

04

以控制地址、平台、架构、channel 为核心维度，适合持续迭代中的轻量 fleet 管理。
