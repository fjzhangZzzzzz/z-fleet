Agent 安装

选择目标平台和发布通道后，系统会直接生成安装命令。架构检测、安装包解析和版本选择都由脚本与后端自动完成。

准备就绪

安装步骤

最少输入，直接执行

  1. 选择平台决定输出 Linux 或 Windows 的安装脚本命令。
  2. 选择发布通道按稳定、候选或开发通道获取默认发布版本。
  3. 生成安装命令系统自动创建一次性注册凭证，并拼出完整命令。
  4. 在目标机器执行脚本会检测架构、下载对应安装包并启动 Agent。

安装脚本会在目标机器上自动识别架构，并向后端请求该平台和通道下的默认安装包。
